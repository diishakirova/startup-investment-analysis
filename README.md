#  Анализ инвестиций в стартапы  

Исследовательский проект по анализу финансирования стартапов на основе исторических данных о компаниях, инвестиционных раундах и возвратах средств.  

---

##  Цель проекта  

Подготовить данные и провести исследовательский анализ, чтобы:  

- изучить структуру финансирования стартапов  
- проанализировать динамику инвестиций по годам  
- определить наиболее распространённые типы финансирования  
- оценить возврат инвестиций  
- сформулировать рекомендации для инвестиционной стратегии  

---

##  Задачи исследования  

В рамках проекта были выполнены:  

**Предобработка данных:**  
- обработка пропусков  
- удаление дубликатов  
- проверка корректности типов данных  
- приведение данных к единому формату  

**Аналитическая часть:**  
- сегментация компаний по срокам финансирования  
- анализ распределения инвестиций и выявление выбросов  
- исследование структуры и типов финансирования  
- анализ динамики инвестиций по годам  
- анализ возвратов средств  
- формирование итоговых выводов и рекомендаций  

---

##  Данные  

В работе использованы два датасета:  

### cb_investments.csv  
Содержит информацию о:  

- компаниях  
- раундах финансирования  
- объёмах инвестиций  
- географии компаний  
- типах финансирования  

### cb_returns.csv  
Содержит данные о возвратах инвестиций по годам и типам финансирования.
